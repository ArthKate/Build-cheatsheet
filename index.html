CSS Cheatsheet
Name Attributes Code snippet Description
Picture N/A 
          <picture>
            <source media="(min-width:650px)" srcset="img_pink_flowers.jpg>
            <source media="(min-width: 1200px)" srcset="medium.jpg>
            <img src="default.jpg" alt="example image">
          </picture>
This tag ensures that images render quickly and fit properly across devices with different screen sizes
Figure src, figcaption 
            <figure>
              <img src="sample.png" alt="Image within an html tag">
              <figcaption>Fig.1 - The image, html figure tag and figure tag Attributes
              </figcaption>
            </figure>
This tag represents self-container elements such as diagrams, images, photos, infographics related to the main flow of the content
iframe src, title 
              <iframe src="https://www.example.com" width="600" height="400" title="Example Website"></iframe>
This tag stands for inline tag and its used to embed another html document into the current one. It can be useful for display content like videos, maps, forms or even webpages within a section of your page.
audio src 
                <audio controls>
                  <source src="audio-file.mp3" type="audio/mpeg">
                  <source src="audio-file.ogg" type="audio/ogg">
                  Your browser does not support the audio tag.
                </audio>
This tag is used to embed audio files in an html document. It supports multiple audio formats like mp3, ogg, wav, etc. The controls attribute adds audio controls like play, pause, and volume.
video src 
            <video width="600px" height="400px">
              <source src="movie.mp4" type="video/mp4">
              <source src="movie.ogg" type="video/ogg">
              Your browser does not support the video tag.
            </video>
This tag enables one to embed video files into the html document. It supports multiple video formats like mp4, avi, mov, webm. The control attribute adds video controls, whereas the src attribute indicates the video source file location.
table N/A 
                <table>
                  <tr>
                  <th>Header 1</th>
                  <th>Header 2</th>
                  </tr>
                  <tr>
                  <td>Data 1</td>
                  <td>Data 2</td>
                  </tr>
                </table>
This tag enables one to create a table in an html document. The table tag is used to define the table itself, the tr tag is used to define the table row, the th tag is used to define the table header, and the td tag is used to define the table data.
